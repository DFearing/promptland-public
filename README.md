# promptland-public

A day-by-day public mirror of [Promptland](https://github.com/DFearing/promptland) (private).

Each PR's title is a Pacific-time development date and its description is a detailed summary of everything that landed on `main` of the private repo that day. The squash-merge of each PR replays that day's net diff onto this public `main`.
